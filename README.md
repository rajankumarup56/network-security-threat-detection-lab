# Network Security & Threat Detection Lab

This project is a hands-on cybersecurity lab focused on analyzing and detecting
network-based threats.

## Tools Used
- Linux
- Wireshark
- Nmap
- Bash

## Focus Areas
- Network monitoring
- Traffic analysis
- Log analysis

## What Was Done
- Performed network scanning using Nmap
- Analyzed network traffic using Wireshark
- Documented threat indicators and logs
- Applied defensive security fundamentals

## Outcome
Gained practical experience in real-world network security scenarios and
understood how threats can be detected at the network level.
